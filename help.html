Tomasulo Simulator

por Joab, Natanael, Paulo e Rafael

Como utilizar:



  • Selecione um exemplo pronto e aperte carregar
  • Ou, de acordo com o formato, descreva as instruções em ordem no quadro de instruções
    1. Após seleciona um exemplo e carregar, ou descrever as instruções, selecione a duração de cada ciclo para cadaa instrução
    2. Com os valores e instruções já definidos selecione o botão "RODAR O ALGORITMO" para que ele inicie
    3. Uma vez que o algoritmo inicia, uma tabela papra o estados das instruções; estado das RS e dos registradores será visível
    4. seleciona "PRÓXIMA ETAPA" para avançar os ciclos das instruções uma por uma, ou, seleciona "RESULTADO FINAL" para que as instruções pule direto para o ciclo final das instruções
  • Caso deseje aplicar outras instruções clique em "COMEÇAR DO ZERO"
